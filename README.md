# Chess Clock

A clean, responsive chess clock built with Vite, React, Tailwind, and shadcn/ui.

## Features

- First-visit setup screen with per-player time controls and increment.
- Optional asymmetric time controls (toggle off “same time”).
- Tap the active timer or press Space to switch turns and apply increment.
- Subtle Web Audio beeps on switch + at 30s and 10s (once each).
- Mute toggle and optional vibration toggle (when supported).
- Light/dark theme switcher.
- Settings button to re-open the setup screen.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
